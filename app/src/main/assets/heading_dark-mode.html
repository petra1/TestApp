WebView

On native Android apps there no heading levels es in HTML. In Android you can only set the appropriate attribute to "true" or to "false". If you switch to the Image screen you can hear the difference between the headings in HTML and the native Android implementation of it.

TalkBack

TalkBack reads the content of locale or external webpage identical to web pages in a browser, because Android using the rendering engine from Google Chrome to display in the app embedded HTML content.
